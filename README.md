7 Days to Die Mods
==================

Copy files like:

```
<7DtD Local Directory>/
    Mods/
        <ModName>/
            Config/
            ItemIcons/
            ModInfo.xml
        <ModName2>/
            Config/
            ModInfo.xml
        ...
```

Or run `copy-mod` with no args.
